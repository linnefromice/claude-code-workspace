# エンタープライズエージェント運用

単一の CLI セッションを超えた運用制御が必要な、クラウドホスト型または継続的に実行されるエージェントシステムにこのスキルを使用します。

## 運用ドメイン

1. ランタイムライフサイクル（開始、一時停止、停止、再起動）
2. オブザーバビリティ（ログ、メトリクス、トレース）
3. 安全制御（スコープ、権限、キルスイッチ）
4. 変更管理（ロールアウト、ロールバック、監査）

## ベースライン制御

- イミュータブルなデプロイメントアーティファクト
- 最小権限の認証情報
- 環境レベルのシークレット注入
- ハードタイムアウトとリトライバジェット
- 高リスクアクションの監査ログ

## 追跡するメトリクス

- 成功率
- タスクあたりの平均リトライ数
- リカバリーまでの時間
- 成功タスクあたりのコスト
- 障害クラスの分布

## インシデントパターン

障害が急増した場合：
1. 新しいロールアウトを凍結
2. 代表的なトレースをキャプチャ
3. 障害ルートを分離
4. 最小限の安全な変更でパッチ
5. リグレッション + セキュリティチェックを実行
6. 段階的に再開

## デプロイメント統合

このスキルと連携するもの：
- PM2 ワークフロー
- systemd サービス
- コンテナオーケストレーター
- CI/CD ゲート
