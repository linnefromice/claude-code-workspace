## 正規サーフェス

- `agent-sort` スキルを直接使用することを推奨します。
- このファイルは互換性のためのエントリーポイントとしてのみ保持してください。

## 委譲

`agent-sort` スキルを適用します。
- 具体的なリポジトリのエビデンスに基づいて ECC サーフェスを分類します。
- 結果は DAILY と LIBRARY に分けて保ちます。
- その後にインストール変更が必要な場合は、ここでインストールロジックを再実装せずに `configure-ecc` へハンドオフします。
